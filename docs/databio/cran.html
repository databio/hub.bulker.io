databio/cran

Home / databio / cran

R environment for CRAN package checking. Uses rocker/tidyverse for both R-release and R-devel. Site-library mounts persist installed packages across container runs, including during R CMD check --as-cran.

Install

bulker activate databio/cran:1.0.0
